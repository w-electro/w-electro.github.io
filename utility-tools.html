أدوات متنوعة مجانية

مجموعة أدوات مفيدة للاستخدام اليومي - توليد أرقام، معالجة نصوص، وتدوين ملاحظات.

كيفية استخدام الأدوات:

مولد الأرقام العشوائية:

  1. حدد الحد الأدنى والأقصى للأرقام
  2. اختر عدد الأرقام المطلوبة
  3. اختر إذا كنت تريد أرقام فريدة (بدون تكرار)
  4. اضغط "توليد" وانسخ النتائج

حذف الأسطر المكررة:

  1. الصق النص (سطر لكل عنصر)
  2. اختر الخيارات (تجاهل الحالة، المسافات، الأسطر الفارغة)
  3. اضغط "حذف المكررات"
  4. راجع الإحصائيات وانسخ النتيجة

مقارنة النصوص:

  1. أدخل النص الأول في الحقل الأيسر
  2. أدخل النص الثاني في الحقل الأيمن
  3. اختر خيارات المقارنة
  4. اضغط "مقارنة" لرؤية الفروق

ملاحظات سريعة:

  1. اكتب ملاحظاتك في المربع
  2. يتم الحفظ تلقائياً في متصفحك
  3. يمكنك تحميلها كملف نصي
  4. الملاحظات تبقى حتى لو أغلقت المتصفح
